## 十、算法处理逻辑说明

本分析报告基于以下算法逻辑生成：

### 10.1 关键词分层逻辑

```
1. 高相关词 (High Relevance)
   ├── P0-套装意图 + 通用词 → Tier 1 精准匹配
   ├── P1-单品意图 + 通用词 → Tier 2 短语匹配
   └── 品牌词 → 品牌截流（特殊处理）

2. 弱相关词 (Weak Relevance)
   └── 月搜索量 >= 50,000 → Tier 3 广泛匹配

3. 看似相关实则无关 (False Positive)
   └── 提取冲突属性词根 → 全局否定词库

4. 完全无关 (Irrelevant)
   └── 提取高频词根 → 全局否定词库
```

### 10.2 流量隔离原则

```
原则1：一个搜索词在同一时间只能在一个广告组中处于"激活"状态
原则2：流量从上层漏斗向下层漏斗流动，不允许反向抢夺
原则3：每层漏斗必须否定上层所有关键词
原则4：广泛匹配层必须否定所有已知的无效词根
```

### 10.3 N-Gram 降噪原则

```
原则1：不盯着每一个独立的搜索词，要看它们共有的特征
原则2：统计"1点击0转化"搜索词中的高频词根
原则3：一次性短语否定高频无效词根，实现"面"的防御
原则4：定期更新否定词库，持续优化流量质量
```
